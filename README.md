## Installation

1. Clonez le dépôt.
2. Accédez au dossier du projet :
   ```bash
   cd chevre-choux
   ```
3. Installez les dépendances :
   ```bash
   npm install --legacy-peer-deps

## Lancement du projet

```bash
npm start
```

L'option `--openssl-legacy-provider` a été intégrée directement dans les scripts du `package.json` pour assurer la compatibilité avec les versions récentes de Node.js (v17+).
